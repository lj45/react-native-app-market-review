# react-native-app-market-review

Review on App Store or Android market directly in your React Native app

## Installation


```sh
npm install react-native-app-market-review
```


## Usage


```js
import { startToComment,startToDetail } from 'react-native-app-market-review';

// ...
startToComment
```
